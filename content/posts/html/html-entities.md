## Html Entities란?
HTML 문서를 코딩할때 태그와 혼용되는 것을 막기위해서 브라우저 상에 예약된 문자(Entities)를 사용한다.

## Entities의 종류

| **&gt;** | 많은 | `&gt;` |
| **&amp;** | and | `&amp;` |
| **&quot;** | 쌍 따옴표 | `&quot;` |

| **&pound;** | 파운드 | `&pound;` |
| **&yen;** | 엔 | `&yen;` |
| **&euro;** | 유로 | `&euro;` |
| **&copy;** | copyright | `&copy;` |
| **&reg;** | 등록상표 | `&reg;` |
